An AI Voice Stress Detection system that records and or inputs voice to detect and analyze the frequencies.
This project aims to give a dedicated score on how one sounds while conversing, wheather it be in interviews or 
in normal conversations
This project uses the following langauges and frameworks

Python: for the Back-end
Frameworks:
fastapi
uvicorn[standard]
python-multipart
numpy
scipy
librosa
scikit-learn
joblib
soundfile
pydantic

NextJS, Typescript, TailwindCSS: for the Front-end
Frameworks: 
Framer Motion
React

This project was made as an Mini Project for College submission, and will be further updated by me to 
work as a browser extension for easier use.
